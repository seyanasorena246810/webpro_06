# データ管理システム (3-in-1)

## 概要
「惑星」「食品添加物」「絶滅危惧種」の3つの異なるカテゴリデータを一括管理できるシステムです。

## 実行方法
1. [cite_start]`npm install` で依存関係をインストール [cite: 64]
2. [cite_start]`node app5.js` を実行 [cite: 72]
3. [cite_start]ブラウザで `http://localhost:8080/` にアクセス [cite: 101]

## 注意事項
- [cite_start]データはメモリ(変数)上に保存されるため、サーバーを停止するとリセットされます [cite: 78]。
- [cite_start]全ての操作はIDを基準に行われます [cite: 47, 160, 206]。
